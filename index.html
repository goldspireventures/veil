How teams get started

  1. Admin creates a team → receives a join code and admin key.
  2. Members install the Veil extension and join with work email.
  3. Everyone pastes or highlights secrets in Outlook / Gmail → secures inline → sends as normal.

IT can push the extension and team passphrase via Chrome or Edge policy — no manual join required.

Security & privacy

  • Encryption runs locally in the browser (AES-GCM + PBKDF2).
  • Veil’s cloud API stores org membership and token ciphertext — not your secrets or passphrases.
  • Security events log metadata only (no matched content).
