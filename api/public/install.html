Personal use

  1. Install from the store button above.
  2. Open Veil → Personal → Quick setup (or choose your own passphrase).
  3. Paste or highlight secrets in Gmail, Outlook, or forms — use Secure or the Veil copilot.

Team members

  1. Install Veil from the store or your IT catalog.
  2. Open the popup → Team → enter join code and work email → Connect.
  3. Cloud teams: passphrase syncs on join — refresh Outlook or Gmail. Self-hosted: use the passphrase your admin shares.
  4. Veil copilot catches secrets on paste; stays quiet on signup forms.

Recipients without Veil

When you secure a message for someone outside your org, Veil can embed a link to the hosted unlock page. They enter the passphrase you shared out of band — decryption runs in their browser; nothing is uploaded.

For IT administrators

Full step-by-step guide: Admin guide (GitHub) · Getting started

  1. Create your team — pick company type; default policy pack is applied automatically.
  2. Sign in at Admin — follow the Setup guide on the Overview tab.
  3. Access — join codes, company default pack, pack library (enable Finance etc. for sub-teams).
  4. Settings → Sub-teams — assign stricter packs to departments (e.g. Finance at a tech company).
  5. People — add emails, assign sub-teams, view connected browsers.
  6. Optional: Security — SIEM webhook, export events (metadata only).

Developer / pilot install

  1. Run npm run package in the Veil repository.
  2. Chrome or Edge → Extensions → Developer mode → Load unpacked → extension/dist.
  3. Reload after each update.

Supported browsers

Chrome 128+, Microsoft Edge 128+, Firefox 128+ (Gecko extension ID required).

Each browser profile keeps its own Veil setup. Join your team separately in Edge and Chrome if you use both.
