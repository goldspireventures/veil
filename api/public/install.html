Install Veil

Browser extension for Chrome and Microsoft Edge.

For team members

  1. Install Veil from your organization’s software catalog, Chrome Web Store, or Edge Add-ons (link from your admin).
  2. Open the Veil popup → choose Team → enter your join code and work email.
  3. Save your team passphrase when prompted.
  4. Refresh Outlook or Gmail, then paste or highlight a secret to see Veil copilot.
Join with a code

For IT administrators

Deploy via Chrome / Edge managed policy with extension ID and team JSON policy. Members skip manual join when setupComplete and teamPassphrase are pushed.

See the extension docs: extension/docs/ENTERPRISE.md in the Veil repository.

Developer / pilot install

  1. Run npm run package in the Veil repository.
  2. Chrome or Edge → Extensions → Developer mode → Load unpacked → select extension/dist.
  3. Reload the extension after each update.

Supported browsers

Chrome 128+, Microsoft Edge 128+, and Firefox 128+ (with Gecko extension ID configured).

Each browser profile keeps its own extension login. Join your team separately in Edge and Chrome if you use both.
